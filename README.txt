# Minetest-Mod-Sieve
This is a simple mod for the open source game "Minetest".
Feel free to use/add your own textures (I used 32x32).
This mod is a work in progress, all blocks work with some bugs.
  Craft Items
  Hand Sieve
  Auto Sieve

Acceptable Items to process:
  "default:sand"
  "default:gravel"
  "default:desert_sand"
  "default:dirt"
