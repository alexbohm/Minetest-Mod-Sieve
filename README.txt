# Minetest-Mod-Sieve
This is a simple mod for the open source game "Minetest".
Feel free to use/add your own textures (I used 32x32).
This mod is still in a non-working state, only the hand sieve works.
Works:
  Hand Sieve
  Crafting items up to hand sieve
Partially-Functional
  Auto Sieve --no fuel consumption, delay time, or repitition for more then one block of a material.
  
Acceptable Items to process:
  "default:sand"
  "default:gravel"
  "default:desert_sand"
  "default:dirt"
